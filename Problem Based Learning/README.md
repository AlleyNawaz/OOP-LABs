# Problem Based Learning (PBL)

This directory contains Problem Based Learning assignments focused on real-world scenarios and applications.

## Structure

Each PBL assignment should be organized in its own subdirectory.

### Example:
```
Problem Based Learning/
├── PBL1/
│   ├── problem_statement.md
│   ├── solution/
│   └── README.md
```

## Guidelines

- PBL assignments focus on applying OOP concepts to real-world problems
- Include a clear problem statement and analysis
- Document your design decisions and implementation approach
- Test your solutions thoroughly
