# Open Ended Lab

This directory contains open-ended lab assignments that require creative problem-solving and advanced implementation.

## Structure

Each open-ended lab should be organized in its own subdirectory with descriptive names:
- `OEL_01/`
- `OEL_02/`
- etc.

## Guidelines

- Focus on demonstrating advanced OOP concepts
- Include comprehensive documentation
- Provide clear explanation of design decisions
