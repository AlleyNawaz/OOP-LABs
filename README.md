# OOP-LABs

This repository contains all coursework for the Object-Oriented Programming (OOP) course, including:
- Lab Tasks
- Open Ended Lab assignments
- Problem Based Learning activities
- Course Project

## Repository Structure

```
OOP-LABs/
├── Lab Tasks/              # Regular weekly lab tasks
├── Open Ended Lab/         # Open-ended lab assignments
├── Problem Based Learning/ # PBL activities and assignments
└── Project/                # Main OOP course project
```

## Purpose

This repository serves as a centralized location for organizing and tracking all OOP coursework throughout the semester. Each directory contains its own README with specific guidelines and structure information.

## Getting Started

1. Navigate to the relevant directory for your assignment
2. Follow the structure and guidelines provided in each directory's README
3. Ensure code follows OOP principles and best practices
4. Include proper documentation and comments

## Contributing

When adding new work:
- Follow the naming conventions specified in each directory
- Include clear comments and documentation
- Ensure code is properly tested
- Update relevant README files if needed
